# Nostrify

Framework for Nostr on Deno and web.

Bring your projects to life on Nostr. 🌱

**Documentation: [nostrify.dev](https://nostrify.dev)**

## Development

### Running Tests

```bash
pnpm i -r
pnpm test
```

### Publishing Packages

This project uses Changesets for managing package versions and releases.

```bash
pnpm changeset
pnpm pkgs:version
pnpm pkgs:publish
```
